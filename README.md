## 프로젝트

### 기능 요구사항
1. 맛집 검색
    1. 다양한 플랫폼에서 조회가능
    2. 검색조건(정확도 순, 리뷰 개수 순)으로 조회 가능
2. 맛집 등록 (자체 관리)
3. 맛집 수정 (자체 관리)
4. 맛집 삭제 (자체 관리)
5. 맛집 검색 -> 키워드 저장
6. 키워드 조회
